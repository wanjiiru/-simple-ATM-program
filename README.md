Description
This is is a python application mimicks a simple ARM

The user will be met with the following commands while signing in:

1. Balance
2. Withdraw
3. Deposit
4. Quit

Prerequiites
- Python 3.6 needed
Set-up and Installation
- Clone the Repo
- Install python 3.6
- Run `atm.py` 
- Run tests on `python -m unittest


Technologies used
- Python 3.6
